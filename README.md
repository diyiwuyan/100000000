# 花光一个亿

5 分钟暴富消费挑战网页小游戏。

## 玩法

1. 创建名字、性别和暴富身份。
2. 等待账户从 0 入账到 1 亿。
3. 在 5 分钟内购买豪宅、豪车、公益、奢侈品、餐饮、艺术收藏、终极体验等项目。
4. 最后用「撒钱开心」清空尾款。
5. 查看称号、用时、消费构成，并生成分享图。

## v1.1 亮点

- 爽玩模式 / 完整模式
- WebAudio BGM 与消费音效
- 连买反馈、震屏、金额飞字
- 随机事件：拍卖会、红毯派对、公益翻倍、车库清仓
- 隐藏挑战：速度、公益占比、模块覆盖、连买
- 更完整的分享战绩卡
- 新增豪车车库、豪宅内景、红毯派对、撒钱红包雨等图片资产
