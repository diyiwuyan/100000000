# 花光一个亿

5 分钟暴富消费挑战网页小游戏。

## 玩法

1. 创建名字、性别和暴富身份。
2. 等待账户从 0 入账到 1 亿。
3. 在 5 分钟内购买豪宅、豪车、公益、奢侈品、餐饮、艺术收藏、终极体验等项目。
4. 最后用「撒钱开心」清空尾款。
5. 查看称号、用时、消费构成，并生成分享图。

## 部署

这是纯静态页面，可直接部署到 GitHub Pages。

推荐仓库名：`spend-100-million`

发布后访问地址通常是：

```text
https://<github-username>.github.io/spend-100-million/
```
